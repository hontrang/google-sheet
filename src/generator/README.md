<h3 align="center">TẠO THƯ VIỆN GS DÙNG TRONG GOOGLE APPS SCRIPT TỪ THƯ VIỆN JAVASCRIPT</h3>

#### 1. Tải thư viện **FULL SOURCE** tại địa chỉ [https://cdnjs.com/libraries/axios/1.11.0]
#### 2. Chạy npm i để cài đặt các thư viện nodejs
#### 3. Copy source code của thư viện vào folder này
#### 4. Thay thế toàn bộ text "lib" thành tên thư viện trong file Main.js. Ví dụ: luxon
![Ví dụ](vd1.png "Ví dụ luxon")
#### 5. Chạy npx webpack --mode production
#### 6. Thư viện được tạo có tên là Code.gs

# Setup Environment
```javascript
npm install -g gulp gulp-exec
npm install -g clasp
clasp login
```

## Start gulp
```javascript
gulp
```

or click start gulp task on left-hand menu

![gulp](./asset/gulp.png "gulp")

## To push code to appscripts
```bash
clasp push
```

## To pull code from appscripts to local
```bash
clasp pull
```

## To execute function
```bash
clasp run [function name]
```
